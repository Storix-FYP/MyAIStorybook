## 🎯 Key Features

### PlantUML Diagrams
- **Best for**: Academic documentation, print quality, detailed customization
- **Format**: Text-based (.puml)
- **Viewing**: PlantUML Online, VS Code extension, Local JAR
- **Export**: PNG, SVG, PDF
- **Advantages**: 
  - Extensive customization
  - Professional appearance
  - Industry standard
  - Excellent for FYP reports

### Mermaid Diagrams
- **Best for**: GitHub/GitLab, web documentation, quick viewing
- **Format**: Text-based (.mmd)
- **Viewing**: GitHub (native), Mermaid Live Editor, VS Code
- **Export**: PNG, SVG, PDF
- **Advantages**:
  - Native GitHub/GitLab rendering
  - No external tools needed
  - Easy markdown integration
  - Great for README files

## 🔧 Quick Start

### Viewing PlantUML Diagrams

**Online (Easiest):**
1. Go to http://www.plantuml.com/plantuml/uml/
2. Copy content from any `.puml` file
3. Paste and view

**VS Code:**
1. Install "PlantUML" extension
2. Open `.puml` file
3. Press `Alt+D`

### Viewing Mermaid Diagrams

**GitHub (Easiest):**
- Just view the `.mmd` files on GitHub - they render automatically!

**Online:**
1. Go to https://mermaid.live/
2. Copy content from any `.mmd` file
3. Paste and view

**VS Code:**
1. Install "Markdown Preview Mermaid Support"
2. Create markdown file with mermaid code block
3. Preview markdown
