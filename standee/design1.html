MyAIStorybook

AI-Powered Personalized Storybook Generator

Motivation

Existing AI image generators produce visually inconsistent characters across multiple scenes. Parents lack the technical skills to write effective prompts. We solve both problems with a zero-prompt, privacy-first pipeline that keeps every child's data local.

Key Objectives

  • Generate coherent multi-page storybooks from a single photo upload
  • Maintain character identity consistency across all illustrations
  • Ensure 100% child-safe content via multi-agent validation
  • Run entirely offline — no cloud dependency, no data leakage

System Design

Multi-Agent Pipeline Diagram

Director → Writer → Reviewer → Editor agents collaborate to produce validated, illustrated narratives.

Technology Stack

Llama 3.1 (Ollama) Stable Diffusion 1.5 IP-Adapter FaceID FastAPI Next.js OpenAI Whisper Coqui TTS

Deliverables

  • Personalized storybook with character-consistent illustrations
  • Synchronized audio narration with emotion-aware voice
  • Interactive AI chatbot for story exploration
  • Print-ready high-resolution PDF export

Future Enhancements

  • Multi-language story generation & narration
  • AR-enabled interactive storybook experience
  • Emotion-based adaptive narrative branching
